***pm.js*** is a fullstack app where ...

The app is built using the following technologies:
- Next.js
- Apollo
- Nexus
- Prisma

- TypeScript

## Prerequisites
To get started, first install ***Node.js*** in your environment.
Then, set up the project by running:

```bash
$ npm install
```
